fn split_path_and_file<'a>(
    dir_name: &str, 
    file_path: &'a str
) -> Option<(String, &'a str)>
Expand description

分割文件所在路径。返回的第一个值是路径，它是新生成的String，复制了需要的字符；第二个值是 slice，指向 file_path 的一部分。

返回的 String 本质上是把 file_path 中除了文件名之外的部分取出，连在 dir_name 后

函数会处理 “./” 和 “../”。如果地址不合法，或者 dir_name 没有以 ‘/’ 结尾，则返回 None
