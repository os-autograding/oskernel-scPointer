打开目录。如果是根目录，特判直接返回 root；否则打开代表目录的 FsDir

因为需要通过 move 传入 root，这个函数只在模块内使用。 如果其他库需要打开目录(作为文件)，需要用 open_file 然后在 flags 里加入 DIR 一项
