Module maturin::signal

Expand description

信号模块，用于 sigaction / sigreturn / kill 等 syscall 信号模块和 task 管理的进程/线程相关，但又相对独立； 且如果单纯作为线程的一部分，容易因为信号发送的任意性导致死锁，因此单独列出来。

目前的模型中，不采用 ipi 实时发送信号，而是由被目标线程在 trap 时处理。因此需要开启时钟中断来保证信号能实际送到
